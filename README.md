# Security Foundations Lab

## Overview
This project simulates a real-world cybersecurity incident involving suspicious login activity and unauthorized file modification.

The lab was built using Kali Linux to apply foundational CompTIA Security+ concepts in a practical scenario.

## Scenario
A user account (jdoe) experienced multiple failed login attempts followed by a successful login. The account then accessed and modified a sensitive payroll file located at /finance/payroll.xlsx.

## Objectives
- Analyze authentication and file access logs
- Identify suspicious behavior
- Apply security concepts to a real-world scenario
- Simulate a basic SOC (Security Operations Center) investigation

## Skills Demonstrated
- Log analysis (Linux environment)
- Security incident investigation
- CIA Triad (Confidentiality, Integrity, Availability)
- AAA (Authentication, Authorization, Accounting)
- Threat, Vulnerability, Exploit, Risk analysis
- Security controls identification

## Tools Used
- Kali Linux
- Vim
- Linux command line

## Project Structure
logs/        -> simulated log data  
scenarios/   -> incident scenario  
analysis/    -> investigation report  
notes/       -> security notes  

## Key Takeaways
This project demonstrates how foundational security concepts can be applied to detect and analyze suspicious activity, similar to tasks performed by SOC analysts.
